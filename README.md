# Hospital Resource Utilization & Patient Outcomes Dashboard

## Overview
This project analyzes hospital resource utilization and patient flow using synthetic healthcare data. It provides insights into department-wise admissions, emergency cases, and patient length of stay.

## Tools & Technologies
- Python
- Google Colab
- Pandas, NumPy
- Matplotlib

## Key Analysis
- Average Length of Stay (ALOS)
- Emergency vs Scheduled Admissions
- Department-wise Admission Volume

## Folder Structure
- notebook/: Google Colab notebook
- data/: Synthetic datasets (CSV)
- screenshots/: Visualization outputs

## Disclaimer
All data used in this project is synthetic and created for educational purposes only.
